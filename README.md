# node-with-ts
## Description

This is a sample node app built in typescript runnig on port 9001.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm start

# building app
$ npm run build
```
## Run app by docker   

First create dir named "mongodb_data" in current project's parent directory for data storage.   
   
```bash
# development
$ docker compose up
```
